## Topic Descriptions
### Data Libraries

NumPy
Provides fast numerical operations using arrays and matrices. It is the foundation of scientific computing in Python.

Pandas
Used for data manipulation and analysis. It introduces DataFrames for handling structured data efficiently.

### Visualization

Matplotlib
A basic plotting library for creating static visualizations like line plots, histograms, and bar charts.

Seaborn
Built on Matplotlib, it provides more advanced and visually appealing statistical plots.

Plotly
Used for interactive and dynamic visualizations, especially useful for dashboards.

Cufflinks
Connects Pandas with Plotly for easy interactive plotting directly from DataFrames.

### Regression Models

Linear Regression
Models the relationship between variables using a straight line. Used for predicting continuous values.

Logistic Regression
A classification model that predicts probabilities using the sigmoid function.

### Model Behavior

Underfitting
Occurs when the model is too simple to capture patterns in data.

Overfitting
Occurs when the model learns noise instead of general patterns, reducing performance on new data.

### Evaluation Metrics

Confusion Matrix
A table that shows classification performance by comparing actual vs predicted labels.

TP: True Positive
TN: True Negative
FP: False Positive
FN: False Negative

AUC-ROC
Measures how well the model separates classes across all thresholds.

### ML Techniques

K-Nearest Neighbors (KNN)
Classifies data points based on the closest neighbors.

Standardization
Scales features to have mean = 0 and standard deviation = 1.

Support Vector Machine (SVM)
Finds the optimal boundary (hyperplane) that separates classes.

Cross Validation
Evaluates model performance using multiple train/test splits.

Kernel Functions
Used in SVM to handle non-linear data (e.g., RBF, polynomial).

### Tree-Based Models

Decision Trees
Split data into branches based on feature values.

Random Forest
An ensemble of decision trees to improve accuracy and reduce overfitting.

### Probabilistic Models

Naive Bayes
A probabilistic classifier based on Bayes’ theorem with independence assumptions.

### Boosting Models

Gradient Boosting Machine (GBM)
Builds models sequentially, each correcting previous errors.

XGBoost
An optimized version of GBM with regularization and high performance.

LightGBM
A faster, memory-efficient boosting algorithm designed for large datasets.

CatBoost
Handles categorical features effectively and reduces preprocessing effort.
